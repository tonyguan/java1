组件------- JDialog (对话框)

相信大家JDialog都很熟悉吧，例如，当你从网上下载一些较大的资料的时候，网站一般会在这时弹出一个对话框，显示文件存储的位置等信息。当你关闭电脑的时候也会弹出一个消息框，等等这些都很好的体现了对话框的应用。

一．JDialog的继承表：

 Javax.Swing.*;

NJdialog

二．Jdialog的构造函数：

Jdialog( )

建立一个non-modal的对话框,没有title 也不属于任何事窗组件

Jdialog(Dialog owner )

建立一个Dialog组件的对话框,为non-modal形式,但没有title

Jdialog(Dialog owner,String title,Boolean modal )

建立一个属于Dialog组件的对话框,可决定modal形式,有title

Jdialog( Frame owner)

建立一个属于Frame组件的对话框,为non-modal形式, 但没有title

Jdialog( Frame owner, String title,Boolean modal)

建立一个属于Frame组件的对话框, 可决定modal形式,有title

  注：上面的说的model是一种对话框的操作模式，当model为true时，代表用户必须结束对话框才能返回到原来所属窗口。当model为false时，代表对话框与所属窗口可以互相切换，彼此之间在操作上没有顺序性。

三．JDialog 在Jbuilder6.0环境下的属性面板：       

name

设置结构面板中JDialog的名称

constraints

对话框在界面中的位置

background

设置对话框的背景颜色

font

设置字体的大小，颜色

forground

设置前景色

label

设置标签的内容


注：在Background（）中，相对应的Jbuilder有两种方法来设置背景颜色。

1．  直接在颜色列表框中选择单一的颜色，在其中有很多种颜色例如有红色，白色，黑色等

单击要选的颜色即可

2．  可在用户颜色设置面板中对颜色进行混合选取，其中有 RGB,HSB两种设置。

（1），在RGB中，它是对基本颜色的选取，有Red,Green,Blue三种可选择的原子颜色，

用户可利用原子颜色来配置自己喜欢的颜色。原子颜色的参数为0~255。

 （2），在HSB中，它是对颜色灰暗程度的选择。用户可对灰暗度进行选择。灰暗度的参数为0~100。

3．  以上所说的颜色（包括颜色的混合）是相对于对话框及字体而言的。

四．Jbuild6.0环境下部分源代码：

package wo;

import java.awt.*;                                             //引入AWT包

import java.awt.event.*;

import javax.swing.*;                                          //引入Swing包

import com.borland.jbcl.layout.*;

public class Frame1 extends JFrame {                            //创建一个Frame1类，继承于Swing组件中的JFrame

  JPanel contentPane;

  JPanel jPanel1 = new JPanel();                                 //创建JPanel

  XYLayout xYLayout1 = new XYLayout();

  public Frame1() {

    enableEvents(AWTEvent.WINDOW_EVENT_MASK);

    try {                                                     //异常处理

      jbInit();

    }

    catch(Exception e) {

      e.printStackTrace();

    }

  }

  private void jbInit() throws Exception  {                            //实例化过程

    contentPane = (JPanel) this.getContentPane();

    this.setSize(new Dimension(400, 300));                           //设置大小

    this.setTitle("对话框实例");                                    //设置标题

    jPanel1.setLayout(xYLayout1);

    contentPane.add(jPanel1, BorderLayout.CENTER);

  }

  protected void processWindowEvent(WindowEvent e) {                 //关闭对话框

    super.processWindowEvent(e);

    if (e.getID() == WindowEvent.WINDOW_CLOSING) {

      System.exit(0);

    }

  }

}
