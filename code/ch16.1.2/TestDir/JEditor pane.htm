二．  在列表框中，有很多重要的内部方法和外部方法，内部方法如下表

Name

设置列表框在结构面板中的名字

Background

设置列表框的背景颜色

font

设置列表框的字体的属性

foreground

 设置列表框的字体的颜色


注：在Background（）中，相对应的Jbuilder有两种方法来设置背景颜色。

1．  直接在颜色列表框中选择单一的颜色，在其中有很多种颜色例如有红色，白色，黑色等

单击要选的颜色即可

2．  可在用户颜色设置面板中对颜色进行混合选取，其中有 RGB,HSB两种设置。

择的原子颜色，

用户可利用原子颜色来配置自己喜欢的颜色。原子颜色的参数为0~255。

 （2），在HSB中，它是对颜色灰暗程度的选择。用户可对灰暗度进行选择。灰暗度的参数为0~100。

3．  以上所说的颜色（包括颜色的混合）是相对于文本域及字体而言的。

三．JBuild6.0环境下源代码如下：

package untitled1;

import java.awt.*;

import javax.swing.*;          //引入SWING包

public class Frame6 extends JFrame {  //创建一个Frame6类，继承于Swing组件中的JFrame

 

  JEditorPane jEditorPane1 = new JEditorPane();  //创建JEditorPane

 public Frame6() {

    try {                //异常处理

     jbInit();

    }

    catch(Exception e) {

      e.printStackTrace();

    }

  }

  private void jbInit() throws Exception {         //实例化

    jEditorPane1.setText("jEditorPane1");         //设置

    this.getContentPane().add(jEditorPane1, BorderLayout.CENTER);//将    添加到面板上

  }

}
